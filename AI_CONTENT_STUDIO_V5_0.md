# AI Content Studio v5.0

## 변경 사항
- 사이드바에서 마케팅 허브를 대시보드 바로 아래로 이동
- AI-CINEBREW 계열은 clean 구조 기준으로 Cine Studio에 통합
- AI-CENEBREW는 스타일/인물 바이블/Export 개념만 부분 채택
- Shorts Script Remix는 Shorts Studio로 통합
- Google Veo3 Short Maker는 Veo Studio로 통합
- Longform Studio는 5단계 파이프라인으로 통합
- Viral Builder는 Viral Studio로 통합
- 중복 API Key 모달/SceneCard/Export 개념은 공통화 방향으로 정리
