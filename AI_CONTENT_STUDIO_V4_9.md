# AI Content Studio v4.9 — Multi Studio Expansion

## 반영 내용

### 1. Business Studio 신규 추가
- `/business`
- 임원 1분 보고서
- 오늘 뭐부터 우선순위 플래너
- 창의적 문제 해결 도구
- 결과 복사/프롬프트 복사

### 2. AI Reports 신규 추가
- `/reports`
- AI 대화 기반 성격 분석 인포그래픽 프롬프트 생성
- MBTI 추정, 사고방식, 강점, 의사결정 스타일 섹션 포함

### 3. Marketing Hub 신규 추가
- `/marketing`
- 각 Studio로 이동 가능한 마케팅 도구 허브
- 카테고리/검색 필터

### 4. Video Studio 업그레이드
- 쇼츠/롱폼/드라마·감성/정보형·브이로그/쇼핑쇼츠 스타일 선택
- 쇼핑쇼츠 원본 URL 입력 필드
- 씬별 이미지 프롬프트 + 영상 프롬프트 + 네거티브 프롬프트
- n8n Scene JSON 복사
- Runway/Kling/Veo 복붙 프롬프트 복사

### 5. Image Studio 업그레이드
- 9:16, 5:4 비율 추가
- 타이포그래피 포스터, 푸드 포스터, 패션 주석, 인포그래픽 빠른 프롬프트 버튼 추가
- 프리미엄 포스터/인포그래픽 템플릿 추가

### 6. Prompt OS 확장
- 임원 보고서 프롬프트
- 우선순위 플래너 프롬프트
- 영상 시나리오 마스터 프롬프트
- 프리미엄 이미지 프롬프트 팩
- AI 대화 기반 성격 분석 리포트 프롬프트

### 7. Amazon SEO 프롬프트 규칙 보강
- HTML ONLY 중심
- 가격 분석, 리뷰 분석, CTA 버튼, Schema, 검색설명 후보 규칙 반영
- 과장 광고/근거 없는 최고 표현 금지

## 참고
- `npm install`은 Prisma postinstall 단계에서 세션 제한으로 완료 확인하지 못했습니다.
- ZIP에는 `node_modules`를 포함하지 않았습니다.
