# AI Content Studio v4.8 Writing + Web Detail Builder

## 업데이트
- `/writing` 안에 블로그 Writing Studio / 웹상세 빌더 탭 추가
- 웹상세 빌더 입력: 브랜드, 제품명, 카테고리, 용량/가격, 판매 플랫폼, 타깃, 성분/스펙, 요청사항, 컨셉
- 카테고리별 기본 스토리보드 자동 생성
- 스토리보드 섹션 추가/삭제/순서변경
- 시즌/무드/이벤트/스토리라인 템플릿 선택
- 제품 이미지/상세페이지/성분표/스펙 문서 업로드 보조
- 상세페이지 생성 API `/api/writing/detail` 추가
- Prompt OS에 웹상세 빌더 프롬프트 3종 추가
- Google News URL 복원 필터 강화: `googleusercontent.com` 최종 노출 방지

## 사용 순서
1. 사이드바 `글쓰기/웹상세` 이동
2. 상단에서 `웹상세 빌더` 선택
3. 제품 정보 입력
4. 카테고리 선택 후 스토리보드 확인/수정
5. 참고자료 업로드
6. `상세페이지 자동 생성`
7. 결과 복사 또는 라이브러리 저장
