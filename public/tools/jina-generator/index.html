사용 가이드

AI 인플루언서 진아 생성기 사용법

  1. 얼굴 고정 — 상단 갤러리에서 진아 레퍼런스 사진을 고르거나, 직접 사진을 업로드하세요. 이 얼굴이 모든 결과물에서 유지됩니다.
  2. 테마 선택 — 요일별 원클릭 테마를 누르면 아웃핏·장소·조명이 한 번에 세팅됩니다. 물론 직접 하나씩 골라도 됩니다.
  3. 세부 조정 — 인물 설정, 착장/장소, 촬영 설정에서 원하는 옵션을 고르세요. 아웃핏·장소는 드롭다운으로 추천을 골라도 되고, 텍스트 칸에 직접 원하는 내용을 입력해도 됩니다.
  4. 제품 착용샷 — "제품 사진으로 착용샷 만들기"를 켜고, 아마존 등에서 캡처한 옷/신발/가방 이미지를 올리면 진아가 그 제품을 그대로 착용한 모습을 생성합니다. 나머지 코디는 아웃핏 텍스트에 적어주세요.
  5. 생성 — "진아 생성하기 (4장)"을 누르면 같은 설정(같은 착장/장소/포즈 등)으로 살짝 다른 4가지 컷을 동시에 만들어줍니다 (약 15~40초 소요). 마음에 드는 컷을 클릭해서 선택하세요. 일부만 실패하면 해당 칸에서 "다시 시도"를 누르면 됩니다.
  6. 다운로드 — "선택한 컷 ZIP 다운로드"는 고른 1장을, "4장 전부 ZIP 다운로드"는 성공한 컷 전체를 압축 파일(.zip)로 받습니다.
  7. 캡션 — 결과물이 마음에 들면 "캡션 만들기"로 인스타그램 게시글 문구 3종을 받아보세요.

⚠️ API 키가 없으면 이미지 생성이 동작하지 않습니다. 설정 메뉴에서 먼저 키를 등록해주세요.

이 도구로 만든 이미지는 실제 인물이 아닌 AI로 생성된 가상 인플루언서 콘텐츠입니다. 게시할 때는 AI 생성 콘텐츠임을 밝히는 것을 권장합니다.
