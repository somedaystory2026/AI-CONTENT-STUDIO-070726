# 🎬 Veo3 쇼츠 프롬프트 생성기

나라와 도시를 선택하면 Google Veo3용 초고속 여행 쇼츠 프롬프트를 자동으로 생성해주는 도구입니다.

## 🚀 사용 방법

1. `index.html` 파일을 브라우저에서 열기 (서버 불필요, 그냥 더블클릭)
2. 나라 선택 → 도시 선택 (선택사항)
3. 영상 옵션 설정 (관광지 수 / 스타일 / 카메라 무빙)
4. **프롬프트 생성** 버튼 클릭
5. **복사** 버튼으로 Veo3에 바로 붙여넣기

## ⚙️ 옵션

- **관광지 수**: 10 / 15 / 20 / 30곳
- **영상 스타일**: Cinematic / Aerial Drone / Golden Hour / Night Life / Cultural Heritage
- **카메라 무빙**: 다이나믹 / 스무스 / 패스트컷 / 슬로우

## 🛠️ 기술 스택

- 순수 HTML / CSS / JavaScript (프레임워크 없음)
- 외부 라이브러리 없음 — 인터넷 연결 불필요
- 다크모드 자동 지원 (`prefers-color-scheme`)
