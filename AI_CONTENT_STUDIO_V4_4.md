# AI Content Studio v4.4

## 변경 사항

- 사이드바 메뉴 순서 변경: 대시보드 → 음악 AI → 뉴스 AI
- 뉴스 수집 화면에서 국가/카테고리 선택 시 기존 수집 목록도 즉시 필터링
- 검색어 + 국가 + 카테고리 필터 동시 적용
- 필터 변경 시 화면에 보이지 않는 기사 선택값 자동 정리
- 뉴스 페이지 localStorage 초기 로딩 방식을 수정해 Hydration mismatch 가능성 완화

## 사용 방식

1. 뉴스 AI에서 국가를 선택합니다.
2. 이미 수집된 기사 목록이 선택 국가 기준으로 바로 분류되어 표시됩니다.
3. 카테고리를 선택하면 국가 + 카테고리 기준으로 다시 좁혀집니다.
4. 최신 뉴스 수집하기를 누르면 선택한 국가/카테고리 기준으로 새 기사만 추가됩니다.
